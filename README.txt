PROYECTO: BOOST – Entrenador personal virtual

📁 Estructura del proyecto:
- backend/ → servidor simulado con json-server
- frontend/ → app en React
- mocks/, docs/ → carpetas de apoyo

▶️ Cómo correr el proyecto:

1. Terminal 1 – Backend
cd backend
npm install
npm start
→ Disponible en: http://localhost:30x01/mensaje

2. Terminal 2 – Frontend
cd frontend
npm install
npm start
→ Disponible en: http://localhost:3000

📡 Conexión:
El frontend se conecta al backend y muestra un mensaje de prueba ("Hola desde la API").

💡 Esto demuestra que la base del proyecto está funcionando correctamente y puede escalarse con rutinas, login, etc.
